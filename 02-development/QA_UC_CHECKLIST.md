# QA — ручной прогон UC-1…UC-5 (QA-01)

> MVP: Flutter → **реальный REST API**. Push (UC-6) — Phase 2.
>
> **OTP в docker/local dev:** `1234` (профиль `docker`, см. `OtpConfig.kt`).

## Подготовка

```bash
cp .env.example .env
docker compose up -d --build
curl http://localhost:8080/health
```

Mobile (эмулятор Android):

```bash
cd mobile
flutter pub get
flutter run --dart-define=API_BASE_URL=http://10.0.2.2:8080/v1
```

Автоматическая проверка API-сценария: `.\scripts\e2e-smoke.ps1` или `mobile/test/e2e/api_smoke_test.dart` с `RUN_E2E=true`.
