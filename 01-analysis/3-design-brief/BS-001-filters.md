## 1. Назначение и контекст

Шторка фильтрации списка слотов на [SCR-002](SCR-002-slot-list.md). Позволяет сузить выдачу по
дате/периоду, типу тренировки (зона/формат), наличию свободных мест и инструктору (FR-4).

## 2. Пользователь и цель

**Цель:** быстро найти подходящую тренировку без долгого скролла по полному списку.

## 3. Навигация

- **Вход:** SCR-002 → «Фильтры».
- **Выход:** «Применить» / «Сбросить» / swipe-to-close → SCR-002 с обновлённым списком.

## 4. Контент-инвентарь

| Фильтр | Параметр API | Тип UI | Примечание |
|--------|--------------|--------|------------|
| Период дат | `date_from`, `date_to` | Date range picker | Дефолт: сейчас + 7 дней |
| Тип тренировки | `zone_format_type` | Мультивыбор чипов | novice / experienced |
| Только свободные | `only_available` | Переключатель | `free_seats > 0` |
| Инструктор | `instructor_id` | Мультивыбор списка | Справочник из API |

## 6. Компоненты и поведение

- **«Применить»** — закрывает шторку, SCR-002 перезагружает список с параметрами; индикатор фильтров на SCR-002.
- **«Сбросить»** — сбрасывает к дефолту (7 дней, без остальных фильтров) → UC-2 A2.
- Период шире 7 дней — UC-2 A1.
- Снек при применении **не показывается** (foundations §6.1).

## 7. Состояния

| Состояние | Поведение |
|-----------|-----------|
| **Loading инструкторов** | Скелетон списка инструкторов; остальные фильтры доступны |
| **Error загрузки инструкторов** | Сообщение + «Обновить» в блоке инструкторов |

## 8. Критерии приёмки

**AC-001** — Все четыре группы фильтров (дата, тип, свободные места, инструктор) представлены.

**AC-002** — «Сбросить» возвращает дефолтную выдачу на 7 дней.

**AC-003** — После «Применить» SCR-002 показывает отфильтрованный список.
