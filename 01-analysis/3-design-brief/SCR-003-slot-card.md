## 1. Назначение и контекст

Экран показывает **все параметры одной тренировки (слота)** для принятия решения о записи.
Промежуточный шаг: [SCR-002](SCR-002-slot-list.md) → SCR-003 → [SCR-004](SCR-004-booking.md).

## 2. Пользователь и цель

- **Цель пользователя:** убедиться, что тренировка подходит (время, формат, инструктор, цена, места, прокат).
- **US-4:** полная карточка перед записью.

## 3. Навигация

- **Вход:** тап по карточке в SCR-002.
- **Выход:** «Записаться» → SCR-004; «Назад» → SCR-002.
- Таб-бар скрыт.

## 4. Контент-инвентарь

| # | Поле | Что показываем | Обяз. |
|---|------|----------------|:-----:|
| 1 | `start_at` | Дата и время старта | Да |
| 2 | `zone_format.name` | Название зоны/формата | Да |
| 3 | `zone_format.type` | Тип (новичковый / опытный) | Да |
| 4 | `zone_format.duration_min` | Длительность (~90 мин) | Нет* |
| 5 | `instructor_info.name` | Инструктор | Да |
| 6 | `price` | Цена за место | Да |
| 7 | `total_seats` / `free_seats` | Всего / свободно мест | Да |
| 8 | `free_rental_equipment` | Свободно прокатных комплектов | Да |

\* Показывать, если значение есть в API.

> Контакты инструктора **не показываются** (NFR-9).

## 6. Компоненты и поведение

### CTA «Записаться»
- **Enabled:** `free_seats > 0` и `status = scheduled`.
- **Disabled:** нет мест — подпись «Мест нет» рядом с CTA.
- **Скрыт/disabled:** слот `cancelled` — «Тренировка отменена» (UC-3 E4).

### Прокатный фонд
- Показывается числом `free_rental_equipment`; влияет на выбор в SCR-004, но **не блокирует** CTA
  «Записаться» (можно записаться со своим снаряжением).

## 7. Состояния

| Состояние | Поведение |
|-----------|-----------|
| **Loading** | Скелетон карточки |
| **Content** | Все параметры + CTA |
| **Нет мест** | Content + disabled CTA «Мест нет» |
| **Слот отменён** | «Тренировка отменена скалодромом»; CTA записи недоступен |
| **Error** | Заглушка + «Обновить» |

## 8. Микрокопия

| Контекст | Текст |
|----------|-------|
| CTA | «Записаться» |
| Нет мест | «Мест нет» |
| Отменённый слот | «Тренировка отменена» |

## 9. Критерии приёмки

**AC-001** — Все обязательные поля FR-5 отображены.

**AC-002** — CTA неактивен при `free_seats = 0`.

**AC-003** — При `status = cancelled` запись недоступна.
