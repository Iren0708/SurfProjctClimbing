# Нефункциональные требования

> Этап 2. Как система должна работать: производительность, надёжность, безопасность,
> удобство, ограничения проекта. Частая ошибка — путать ФТ и НФТ.
>
> **Трассировка:** [traceability-matrix.md](traceability-matrix.md).

> **Скоуп — клиентское мобильное приложение и API для него.** НФТ относятся к клиентскому
> приложению; функции инструктора/владельца обеспечиваются существующей инфраструктурой.
