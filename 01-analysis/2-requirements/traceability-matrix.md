# Матрица трассировки требований

> Сквозная трассировка: **Бриф** → **Ответы заказчика (Q)** → **Домен** → **BR** → **FR / NFR** → **US** → **UC**.
> Детальные формулировки — в соответствующих файлах этапа 2.

## Обозначения источников

| Код | Файл | Как читать |
| :-- | :-- | :-- |
| **Бриф §L*N*** | [brief-climbing.md](../0-customer-brief/brief-climbing.md) | Строка *N* в письме Оли (до блока «Уточнение по скоупу») |
| **Бриф §уточн:*R-NNN*** | [brief-climbing.md](../0-customer-brief/brief-climbing.md) | Параграф уточнения по скоупу (R-027, R-008 и т.д.) |
| **Q-*N*** | [customer-questions.md](../1-elicitation/customer-questions.md) | Ответ заказчика на вопрос № *N* |
| **Домен §*раздел*** | [domain-description.md](../1-elicitation/domain-description.md) | Раздел описания домена |

## Цепочка уровней

```text
Бриф + Q  →  Домен  →  BR  →  FR / NFR  →  US  →  UC
```
